### Prerequisites

- GCC >=4.8
- G++ >= 4.8
```
export CXX=g++
```
- SWIG >= 2.0
```
sudo apt-get update
sudo apt-get install swig
```
- Boost >= 1.6
```
sudo apt-get update
sudo apt-get install libboost-all-dev
```
- CMake >= 3.1
```
sudo apt-get update
sudo apt-get install cmake
```
- Current support for kicad_pcb format derived from KiCad v5.1.2
